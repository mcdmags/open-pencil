# Редактирование векторных объектов

Режим редактирования векторных объектов позволяет менять **геометрию** кривой: положение узлов, форму сегментов и рычаги Безье.  
В этом режиме вы работаете именно с контуром, а не с обычными трансформациями объекта.

## Как войти в режим

- Выберите векторный объект инструментом «Выделение».
- Сделайте **двойной клик по кривой**.

После этого включается режим редактирования геометрии выбранного вектора.

## Как выйти из режима

- Нажмите <kbd>Escape</kbd>.
- Или переключитесь на другой контекст редактирования.

## Что меняется в этом режиме

- Обычный bounding box трансформаций для объекта отключается.
- Доступно редактирование узлов, сегментов и рычагов.
- Курсор и взаимодействия не переключаются на resize/rotate у углов bbox.

## Базовые действия

### Перемещение узла

- Потяните опорную точку (узел) мышью.
- Примыкающие сегменты и форма контура обновляются сразу в предпросмотре.

### Редактирование рычага Безье

- Потяните рычаг узла.
- По умолчанию поведение определяется текущим типом композиции рычагов узла.

## Модификаторы при перетаскивании рычага

| Действие | Mac | Windows / Linux |
|----------|-----|-----------------|
| Continuous (Гладкие / Зависимые) | <kbd>Cmd</kbd> + drag | <kbd>Ctrl</kbd> + drag |
| Corner (Независимые рычаги) | <kbd>Option</kbd> + drag | <kbd>Alt</kbd> + drag |
| Фиксация направления (только длина) | <kbd>Shift</kbd> + drag | <kbd>Shift</kbd> + drag |

### Continuous: <kbd>Cmd</kbd>/<kbd>Ctrl</kbd> + drag

- Активный рычаг выстраивается по той же прямой, что и сестринский.
- Меняется только длина активного рычага.
- Удобно для плавного перехода без излома.

### Corner: <kbd>Option</kbd>/<kbd>Alt</kbd> + drag

- Активный рычаг редактируется независимо.
- Сестринский рычаг не двигается.
- Удобно для создания «ломаного» перехода.

### Фиксация направления: <kbd>Shift</kbd> + drag

Для узлов с композициями **Continuous** и **Symmetric**:

- направление пары рычагов фиксируется в положении, которое было **до начала текущего перетаскивания**;
- перетаскивание меняет только длину (или длины, в зависимости от композиции узла).

## Переопределение изгиба через перетаскивание узла

При перетаскивании узла с зажатым <kbd>Cmd</kbd>/<kbd>Ctrl</kbd> система выбирает редактируемый рычаг по **направлению примыкания сегмента** к узлу (а не просто по ближайшей соседней точке).  
Это работает и для узлов с несколькими примыкающими сегментами (vector web): после выбора целевой рычаг фиксируется на весь текущий drag.

## Работа Пером внутри режима редактирования

Если активирован инструмент «Перо»:

- **Клик по сегменту** — добавляет новую опорную точку (разрезает сегмент).
- **Клик по концевой точке** открытого контура — продолжает рисование от этой точки.
- **Option/Alt + клик по узлу** — удаляет узел (если топология позволяет).

Подробнее о создании контуров и замыкании: [Инструмент «Перо»](./pen-tool.md).

## Практический сценарий

1. Нарисуйте форму Пером.
2. Двойным кликом войдите в «Редактирование векторных объектов».
3. Подвиньте узлы, чтобы поправить силуэт.
4. Потяните рычаги:
   - с <kbd>Cmd</kbd>/<kbd>Ctrl</kbd> для плавного сопряжения,
   - с <kbd>Option</kbd>/<kbd>Alt</kbd> для независимой правки,
   - с <kbd>Shift</kbd> для изменения только длины.
5. Нажмите <kbd>Escape</kbd>, чтобы выйти.
